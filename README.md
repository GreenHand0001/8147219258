# 项目简介

## 核心功能
- 基于 YOLOv8 的武器检测系统，支持检测 ['Handgun', 'Rifle'] 两类武器
- 提供完整源码、17000 张标注数据集、训练好的模型及训练结果
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 界面功能：实时显示目标位置、目标总数、置信度等信息
- 支持自定义检测参数：置信度阈值、IOU阈值
- 提供检测结果保存和批量检测查看功能
- 包含完整项目运行教程和 UI 界面源文件

## 适用场景
- 安防监控系统的武器检测
- 视频内容安全审核
- 边境安检和公共场所安全监测
- 研究和教育领域的目标检测应用
- 需要实时武器识别的安全管理场景

## 技术特点
- 采用 Pycharm/VSCode + Anaconda 开发环境
- 基于 python3.9、opencv-python、PyQt5、ultralytics、torch1.9 等技术栈
- 提供图形用户界面，操作简便直观
- 支持多格式输入和实时检测结果展示
- 模型已训练完成，配置环境后可直接使用
